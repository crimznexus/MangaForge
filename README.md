# MangaForge

> Under development. Features may be incomplete and behavior can change frequently.

### Android manga/webtoon reader
A fork of Mihon, built with Kotlin + Jetpack Compose.

## Build from source

This project uses Gradle/Android Studio.

Prerequisites:

- JDK 17
- Android SDK (this project targets `compileSdk=36`)

Example commands (Windows):

- `.\gradlew.bat :app:assembleDebug`
- `.\gradlew.bat :app:installDebug`
